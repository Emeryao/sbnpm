# sbnpm

> `SB npm` 不用翻墙也能用的npm  

## Install

With [npm](https://npmjs.org/) installed, run

```batch
npm install -g sbnpm
```

## Usage

The entry command is ***`sbnpm`***  
**JUST** use `sbnpm` like `npm`  
The **ONLY** difference is the `registry` param is always `https://registry.npm.taobao.org`

## Example

```batch
sbnpm install -g ts-starter
```
